# Decathlon Calculator Example Project

A small decathlon calculator project created for a job interview. Contains a back-end front-end and a database.  
Uses Spring Boot and PostgreSQL. Back-end also has tests for the controller.  

## Running the project

Project can be run with docker:
`docker compose up`

For development back-end can be run with Maven/IntelliJ, and front-end can be run with Node `npm run dev`.
